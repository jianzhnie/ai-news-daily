Last30Days Skill

概述

Last30Days Skill 是一个 AI Agent Skill，能够跨 Reddit、X、YouTube、Hacker News、Polymarket 和 Web 进行多源研究，并合成有据可查的摘要。由 mvanhorn 开发。其核心创新在于将预测市场（Polymarket）作为信息源引入研究流程，在时效性分析和事件驱动研究中具有独特优势。

关键贡献

  • 跨平台信息聚合 — 打破单一平台 Agent 的”信息孤岛”问题
  • Polymarket 预测市场纳入 — 将”市场共识”作为独特的信息维度
