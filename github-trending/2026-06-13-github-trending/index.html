今日趋势概览

今日榜单几乎被 AI Agent 基础设施 承包：obra/superpowers 提供系统化的 Agent 工程方法论，addyosmani/agent-skills 给出可落地的生产级技能模板，而 NVIDIA/SkillSpector 则从安全侧补齐了 Agent skills 供应链治理的关键拼图。三者共同指向一个信号：Agent 开发正从”提示词工程”走向”工程化、可审计、可复用的技能体系”。

与此同时，Apple/container 以近 1.5k 日增强势上榜，标志着 Apple 在容器/虚拟化开发者工具上的发力；老牌效率工具 Microsoft/PowerToys 和开源客服平台 chatwoot 持续保持高关注度，说明开发者对”稳定好用的基础设施”仍有强烈需求。AI 安全、容器化、Agent 工程化，构成了今日趋势的三条主线。

★★★ 必关注

obra/superpowers

  • Star: 226,617 total / +931 today
  • 语言: Shell
  • 是什么：一套 Agentic Skills 框架与软件工程方法论，旨在让 AI 编码助手像”有纪律的工程师”一样工作。
  • 为什么火：Jesse Vincent（obra）将 RED-GREEN-REFACTOR TDD、Socratic 设计、系统调试、Git worktree 隔离等工程实践封装为 14 个可自动触发的 skill，填补了 Agent “会写代码但不会工程化” 的空白。正值 Claude Code、Codex、Cursor 等 AI 编码工具爆发期，社区急需一套标准化的 Agent 工作流。
  • 适合谁：使用 Claude Code / Cursor / Codex 进行日常开发、希望把 Agent 从”打字机”升级为”协作工程师”的团队。

addyosmani/agent-skills

  • Star: 57,805 total / +1,507 today
  • 语言: Shell
  • 是什么：Google Chrome 工程负责人 Addy Osmani 整理的生产级 AI 编码 Agent 技能集合。
  • 为什么火：直接对接《Software Engineering at Google》等工程最佳实践，把 Google 级代码审查、测试、重构、架构设计流程转化为 Agent 可调用的 skill。日增 1.5k 说明开发者对”大厂工程规范如何落地到 AI 编码”有强烈需求。
  • 适合谁：前端/全栈开发者、希望在 Agent 工作流中引入企业级工程纪律的技术团队。

NVIDIA/SkillSpector

  • Star: 4,038 total / +809 today
  • 语言: Python
  • 是什么：NVIDIA 开源的 AI Agent skill 安全扫描器，检测漏洞、恶意模式和供应链风险。
  • 为什么火：Agent skills 正在成为新的”软件依赖”，但生态缺乏类似 npm audit 的安全治理。SkillSpector 覆盖 16 类风险（prompt injection、MCP tool poisoning、hidden instructions、excessive agency 等），并提供 SARIF/JSON 输出接入 CI/CD。NVIDIA 推出 Verified Agent Skills 生态的标志性工具。
  • 适合谁：构建或分发 Agent skills 的团队、关注 AI 供应链安全的 SRE / 安全工程师。

apple/container

  • Star: 35,904 total / +1,471 today
  • 语言: Swift
  • 是什么：Apple 官方推出的容器工具，基于轻量级虚拟机（Apple Virtualization Framework）在 Mac 上创建和运行 Linux 容器。
  • 为什么火：专为 Apple Silicon 优化，Swift 编写，补足 Mac 上原生 Linux 容器体验的短板。随着 Apple Silicon 在开发者群体中的普及，以及本地 AI 推理、容器化开发需求增长，Apple 入场容器工具赛道意义重大。
  • 适合谁：Mac 开发者、需要在本地获得接近 Linux 原生容器体验的团队、Apple Silicon 重度用户。

iptv-org/iptv

  • Star: 118,499 total / +650 today
  • 语言: TypeScript
  • 是什么：全球公开 IPTV 频道链接集合，不提供视频文件，只聚合公开可访问的流媒体 URL。
  • 为什么火：作为长期积累的”流媒体入口”项目，持续受益于 cutter-cord 趋势和全球用户对免费流媒体源的需求。今日 +650 日增显示社区活跃度仍在上升。
  • 适合谁：自建家庭媒体中心（如 Jellyfin/Plex/Kodi）的用户、流媒体爱好者。
