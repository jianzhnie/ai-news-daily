月度趋势概览

5 月的 GitHub Trending 由 AI Coding 工具链的全面爆发定义。19 个上榜项目中，AI Coding 相关占 13 个（68%）。三大趋势清晰可见：知识图谱成为 Agent 新基础设施（CodeGraph 3.7 万 star、Understand-Anything 5 万 star）、Agent 记忆层独立成赛道（agentmemory 2 万 star、Claude Code skills 11 万 star）、AI Coding 终端化（CodeWhale 3.6 万 star、oh-my-pi 1 万 star）。Andrej Karpathy 的 Claude Code 技能文件以 16.5 万 star 登顶月榜总星数第一。

★★★ 必关注

multica-ai / andrej-karpathy-skills

  • Star: 165,870
  • 语言: —
  • 是什么：一个 CLAUDE.md 文件，从 Andrej Karpathy 对 LLM 编码缺陷的观察中提炼，用于改善 Claude Code 行为
  • 为什么火：16.5 万 star 是本月最高总量。一个单一配置文件能获得如此关注，说明两个趋势：1) AI Coding 工具的”提示词工程”正在成为独立技能；2) Karpathy 的个人品牌在 AI 工程社区的影响力达到了新高度。本质上是”如何让 Claude Code 写得更好”的标准答案
  • 适合谁：所有 Claude Code 用户

colbymchenry / codegraph

  • Star: 37,949
  • 语言: TypeScript
  • 是什么：预索引的知识图谱，为 AI Coding Agent（Claude Code、Codex、Gemini 等）减少 Token 消耗和工具调用
  • 为什么火：CodeGraph 的走红标志着一个重要转变——AI Coding 工具的优化从”更好的提示词”进入”更好的上下文”。通过预构建代码知识图谱，大幅减少 Agent 需要的 context window。这本质上是为 AI Agent 准备”预消化的代码地图”
  • 适合谁：大型代码库的 AI Coding 用户
